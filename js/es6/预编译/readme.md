# 预编译 发生在函数执行之前 四部曲
1. 创建AO对象（Activation Object）
2. 找形参和变量声明，将变量声明和形参作为AO的属性名，值为undefined
3. 将实参和形参值统一
4. 在函数体里找函数声明，将函数名作为AO对象的属性名，值赋予函数体
